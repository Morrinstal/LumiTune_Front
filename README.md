Frontend part of the **LumiTune** music service.  
Tech stack: **React + TypeScript + Vite**.

## Features
- Music service UI
- Communicates with the backend API (Django REST Framework)
- Fast development experience with Vite (HMR)

## Tech Stack
- React
- TypeScript
- Vite

## Getting Started

### 1) Install dependencies
- bash
- npm install

### 2) Run in development mode
- npm run dev
- The app is usually available at: http://localhost:5173
